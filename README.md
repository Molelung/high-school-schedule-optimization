# 高中课程安排优化设计研究

这是一个关于高中课程安排优化设计研究的PPT展示项目。

## 项目结构

```
├── index.html          # 主入口文件
├── styles.css          # 样式文件
├── script.js           # 交互逻辑
└── slides/             # 幻灯片目录
    ├── 00-ready.html   # 就绪页
    ├── 01-cover.html   # 封面
    ├── 02-toc.html     # 目录
    ├── 03-background.html # 研究背景
    ├── 04-literature.html # 文献综述
    ├── 05-questions.html  # 研究问题
    ├── 06-methods.html    # 研究方法
    ├── 07-analysis.html   # 结果分析
    ├── 08-discussion.html # 结果讨论
    ├── 09-conclusion.html # 研究结论
    ├── 10-reflection.html # 反思与展望
    └── 11-defense.html    # 答辩
```

## 功能特性

- 📖 文献综述翻书效果
- 🎯 研究问题卡片动画
- 💭 研究方法泡泡交互
- 📊 结果分析图表动画
- 📜 研究结论羊皮卷轴
- 🎴 反思与展望卡片堆叠
- 🎯 答辩页自动播放

## 使用方法

直接打开 `index.html` 文件即可查看PPT。

## 部署

本项目已部署到 GitHub Pages。
